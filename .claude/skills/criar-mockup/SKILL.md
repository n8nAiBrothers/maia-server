# Skill: UX/UI Designer (Criar Mockup)

Sempre que o usuário precisar visualizar uma interface antes do desenvolvimento completo, utilize esta skill para gerar mockups.

## Passos para Execução:

1. **Entendimento da Interface**:
   - Qual é o objetivo da tela? Quais componentes são vitais?
   - Defina o layout (Mobile-first ou Desktop-first).

2. **Geração do Mockup**:
   - Utilize as melhores práticas de design (Premium, Moderno, Cores Ricas, Sombras Suaves, Glassmorphism).
   - Você pode gerar um mockup de duas formas:
     a) **Código HTML/Tailwind em arquivo único**: Crie um arquivo HTML contendo o design completo via CDN do Tailwind, focado 100% no visual, para o usuário abrir e visualizar a tela.
     b) **Geração de Imagens**: Se você tiver acesso a uma ferramenta de geração de imagens, gere a representação visual da UI.

3. **Validação de Qualidade**:
   - O mockup deve ser "Pixel-Perfect", simulando uma entrega de produção.
   - Aplique padrões de tipografia moderna (Inter, Roboto, Poppins).

4. **Entrega**:
   - Salve o arquivo (ex: `mockup_dashboard.html`) e forneça as instruções visuais.
