# 🚀 Servidor de IA: Arquitetura & Autonomia

## 1. Infraestrutura Core
* **Host:** Mac M5 Pro (Alta Performance Local).
* **Stack:** n8n + PostgreSQL + Ollama + **Obsidian**.
* **IA Local:** Gemma 4 (Foco em Privacidade e Volume).

## 2. Estratégia Híbrida (AI Routing)
* **Local:** Processamento de dados sensíveis e tarefas volumosas (Custo $0).
* **Cloud:** Claude (Anthropic) e Gemini (Google) para lógica complexa.
* **Gestão:** Otimização automática entre Custo x Performance x Privacidade.

## 3. Papel do Agente (Antigravity)
* **Admin:** Configuração de serviços, gestão de bancos e debug de fluxos.
* **Orchestrator:** Transformar conversas em ações reais no sistema de arquivos.
* **Task Manager:** Gestor de tarefas do grupo dentro do servidor.

## 4. Conectividade & Equipe
* **Remote Access:** Tailscale (VPN) / SSH para trabalho em grupo.
* **Gateway WhatsApp:** Interface simplificada via n8n para a equipe.
* **Obsidian Knowledge Base:** Gestão de conhecimento e notas técnicas.
* **Workspace Sync:** Sincronização automática de entregas (Google Drive).

## 5. Próximos Passos (Terça-feira)

2. Configuração do túnel de acesso remoto.
3. Setup do Gateway n8n <-> Ollama.
